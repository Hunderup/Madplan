🍳 Kokkens Hjørne — Opsætning

For at data gemmes på tværs af enheder kræves en gratis Supabase-database. Opsætningen tager ~5 minutter og gøres kun én gang.

  1. Gå til supabase.com → Opret gratis projekt
  2. Gå til Project Settings → API
  3. Kopiér Project URL og anon public key
  4. Kør dette SQL i SQL Editor: create table if not exists households ( code text primary key, name text, members jsonb default '[]', stores jsonb default '[]', shop_day text default '', data jsonb default '{}' ); alter table households enable row level security; create policy "open" on households for all using (true);

Vil du åbne appen fra et fast link (så login huskes automatisk på din enhed)?

🚀 Netlify Drop — gratis, 1 minut
  1. Gå til app.netlify.com/drop
  2. Træk HTML-filen ind på siden
  3. Du får et fast link, f.eks. https://dit-navn.netlify.app
📁 GitHub Pages — gratis, kræver GitHub-konto
  1. Opret nyt repository → upload HTML-filen som index.html
  2. Settings → Pages → Deploy from branch → main
  3. Link: https://brugernavn.github.io/repo

💡 Tilføj linket til din telefons hjemmeskærm (Del → "Føj til hjemmeskærm") for app-oplevelse uden browser-chrome.
